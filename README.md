## Features

- [x] Printf
- [x] Context switching
- [x] Exception handling
- [x] Memory allocation
- [x] Page tables
- [x] Virtual memory
- [x] Syscalls  
- [x] User mode
- [x] Interactive shell
- [x] Virt-IO basic driver
